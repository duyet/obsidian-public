Restore old photographs to professional Canon EOS R5 quality with vivid colorization.

## The Prompt

```
Restore and enhance this photograph to professional portrait quality
comparable to a Canon EOS R5 capture. Repair all scratches, folds,
stains, discoloration, and missing details with high-precision
reconstruction while preserving the subject's natural facial features,
skin texture, and lighting. Vivid Colourful.
```

## What It Does

| Feature | Description |
|---------|-------------|
| **Professional Quality** | Canon EOS R5 level output |
| **Damage Repair** | Scratches, folds, stains, discoloration |
| **Detail Reconstruction** | Fills in missing details |
| **Feature Preservation** | Maintains natural facial features and skin texture |
| **Vivid Colors** | Bright, saturated colorization |

## Tips

- Specifying camera model (Canon EOS R5) guides quality expectations
- "Vivid Colourful" produces more saturated results than neutral prompts
- Works well for portraits with significant damage
- Preserves ethnic facial features when specified
